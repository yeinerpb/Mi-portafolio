    Mis proyectos

    Durante mi aprendizaje de estas tecnologías e podido desarrollar diferentes aplicaciones. En ellas puedo mostrar mis conocimientos y capacidades. Las tecnologías que utilizo para estas aplicaciones son: Java Script, HTML, CSS, React y Node js.

    Pokedex

    En esta aplicacion se muestran todos los avatares por su id y nombre. También se puede filtrar por tipo. Cada avatar tiene sus características. Para su desarrollo se uso: react, redux, router-dom, css, html y consumo de API.

    Rick and Morty

    Muestran los personajes de la serie y permite buscar por ID, tipo o ubicación. Cada personaje tiene sus respectivas características. Para su desarrollo se uso: react, axios, css y consumo de API.

    Ecommerce

    API back-end creada con NODE JS. En esta API permite por medio de un usuario la creación de productos de un ecommerce. Para su desarrollo se usó: Node.js, Express, Sequelize, PostgreSQL, JWT.
